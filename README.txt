## NAME
    Devin's Game

## DESCRIPTION
Devin's Game is a typing tutor that allows my son to learn how to recognize letters 
as well as find them on the keyboard.

## INSTALL

Must have python installed on users computer
May need to install pygame and pygame-menu libraries

## BASH

pip install pygame
pip install pygame-menu

## RUNNING

Open directory where all game is located. All necessary files are located in the folder
labeled DEVINS_TYPING_TUTOR
Once in the proper directory input the following command in the terminal

python3 devins_game.py 

this will run the full game.

## NOTE

Please do not remove any files from folder as they are necessary for the running of the game.
